When Functions Multiply

The Power Rule is great until you hit two functions multiplied together. You might think you can just multiply their derivatives. You can't.

Let's say you need to differentiate $f(x) = x^2 \times (3x+1)$.

The greatest trap in all of calculus is thinking the answer is the derivative of the first part times the derivative of the second part. If you do that, you get $2x \times 3 = 6x$.

But wait... if we expand the original brackets first, $f(x) = 3x^3 + x^2$. If we use the Power Rule on that, we get $9x^2 + 2x$.

$6x \neq 9x^2 + 2x$. The naive way failed.

Why? Because of geometry. Let's look at how area grows.

The Product Rule Formula

If a function is made of two pieces multiplying, $f(x) = u(x)v(x)$, then:

$$f'(x) = u'(x)v(x) + u(x)v'(x)$$

Memory Trick: "Left d-Right plus Right d-Left"

Applying the Product Rule

Worked Solutions

🟢 Easy: Basic Brackets

Question: Differentiate $f(x) = x^2(3x + 4)$

1. Identify the pieces: $u = x^2$ and $v = 3x + 4$

2. Differentiate both: $u' = 2x$ and $v' = 3$

3. Apply formula ($u'v + uv'$):

$$f'(x) = (2x)(3x + 4) + (x^2)(3)$$

4. Expand and simplify:

$$f'(x) = 6x^2 + 8x + 3x^2 = 9x^2 + 8x$$

🟡 Medium: Two Polynomials

Question: Differentiate $f(x) = (2x^2 - 1)(x^3 + 5x)$

1. Identify: $u = 2x^2 - 1$ and $v = x^3 + 5x$

2. Differentiate: $u' = 4x$ and $v' = 3x^2 + 5$

3. Apply formula:

$$f'(x) = (4x)(x^3 + 5x) + (2x^2 - 1)(3x^2 + 5)$$

4. Expand:

$$f'(x) = (4x^4 + 20x^2) + (6x^4 + 10x^2 - 3x^2 - 5)$$ $$f'(x) = 10x^4 + 27x^2 - 5$$

🔴 Hard: Fractional Powers

Question: Differentiate $f(x) = \sqrt{x}(4x^2 - 3)$

1. Rewrite roots as powers: $u = x^{1/2}$ and $v = 4x^2 - 3$

2. Differentiate: $u' = \frac{1}{2}x^{-1/2}$ and $v' = 8x$

3. Apply formula:

$$f'(x) = (\frac{1}{2}x^{-1/2})(4x^2 - 3) + (x^{1/2})(8x)$$

4. Expand carefully (add powers when multiplying):

$$f'(x) = 2x^{3/2} - \frac{3}{2}x^{-1/2} + 8x^{3/2}$$ $$f'(x) = 10x^{3/2} - \frac{3}{2}x^{-1/2}$$
