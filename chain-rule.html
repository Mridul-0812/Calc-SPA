Functions Inside Functions

What happens when a function swallows another function whole? You can't just use the Power Rule. You need to break the chain.

Imagine you have the function $y = (3x + 1)^2$.

You could expand the brackets to get $y = 9x^2 + 6x + 1$, and then differentiate it using the Power Rule to get $\frac{dy}{dx} = 18x + 6$. That works fine.

But what if you had $y = (3x + 1)^{100}$? Expanding that would take you the rest of your life. We need a shortcut. We need a way to deal with composite functions (a function nested inside another function).

The Chain Rule

There are two ways to write this, depending on your preferred notation:

Leibniz Notation
$$\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$$
Function Notation
$$f'(g(x)) \times g'(x)$$

Memory Trick: "Differentiate the outside, leave the inside alone, then multiply by the derivative of the inside."

Applying the Chain Rule

Worked Solutions

🟢 Easy: Basic Brackets

Question: Find $\frac{dy}{dx}$ for $y = (2x + 5)^6$

1. Outside function is $(\text{stuff})^6$. Inside function is $u = 2x + 5$.

2. Differentiate the outside: Bring down the 6, power becomes 5.

$$6(2x + 5)^5$$

3. Multiply by the derivative of the inside ($2$):

$$\frac{dy}{dx} = 6(2x + 5)^5 \times 2$$

4. Simplify the numbers on the outside:

$$\frac{dy}{dx} = 12(2x + 5)^5$$

🟡 Medium: Square Roots

Question: Find $f'(x)$ for $f(x) = \sqrt{x^2 + 1}$

1. Rewrite the root as a fractional power:

$$f(x) = (x^2 + 1)^{1/2}$$

2. Differentiate outside (power $\frac{1}{2}$ comes down, subtract 1 to get $-\frac{1}{2}$):

$$\frac{1}{2}(x^2 + 1)^{-1/2}$$

3. Multiply by derivative of inside ($2x$):

$$f'(x) = \frac{1}{2}(x^2 + 1)^{-1/2} \times 2x$$

4. Simplify ($\frac{1}{2} \times 2x = x$):

$$f'(x) = x(x^2 + 1)^{-1/2} \text{ or } \frac{x}{\sqrt{x^2+1}}$$

🔴 Hard: Fractions and Negative Powers

Question: Find $\frac{dy}{dx}$ for $y = \frac{1}{(3x^2 - x)^3}$

1. Do not use the Quotient Rule! It's much faster to rewrite using a negative power:

$$y = (3x^2 - x)^{-3}$$

2. Differentiate outside (bring down $-3$, power becomes $-4$):

$$-3(3x^2 - x)^{-4}$$

3. Multiply by derivative of inside (put it in brackets!):

$$\frac{dy}{dx} = -3(3x^2 - x)^{-4} \times (6x - 1)$$

4. Clean it up (move the negative power back to the bottom):

$$\frac{dy}{dx} = \frac{-3(6x - 1)}{(3x^2 - x)^4}$$
