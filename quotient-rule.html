When Functions Divide

You now know what to do when functions multiply. But what if one function is stacked on top of another as a fraction? Enter: The Quotient Rule.

Consider the function $f(x) = \frac{x^2}{3x - 2}$.

Just like the Product Rule, you cannot simply divide the derivatives. If you try to say $f'(x) = \frac{2x}{3}$, you will get it wrong every single time.

Because fractions are essentially just multiplication in disguise (e.g., dividing by $x$ is the same as multiplying by $x^{-1}$), the Quotient Rule looks suspiciously similar to the Product Rule. But it has a notorious trap: a minus sign.

The Quotient Rule Formula

If $f(x) = \frac{u(x)}{v(x)}$, then the derivative is:

$$f'(x) = \frac{v(x)u'(x) - u(x)v'(x)}{[v(x)]^2}$$

Memory Rhyme: "Low d-High minus High d-Low, square the bottom and away we go!"

Applying the Quotient Rule

Worked Solutions

🟢 Easy: Linear over Linear

Question: Differentiate $f(x) = \frac{2x}{x - 3}$

1. Identify: $u = 2x$ (High) and $v = x - 3$ (Low)

2. Differentiate: $u' = 2$ and $v' = 1$

3. Apply "Low d-High minus High d-Low, over Low squared":

$$f'(x) = \frac{(x-3)(2) - (2x)(1)}{(x-3)^2}$$

4. Expand the top ONLY. (Leave the bottom squared, it's tidier!):

$$f'(x) = \frac{2x - 6 - 2x}{(x-3)^2} = \frac{-6}{(x-3)^2}$$

🟡 Medium: Polynomials

Question: Differentiate $f(x) = \frac{x^2 + 1}{3x}$

1. Identify: $u = x^2 + 1$ and $v = 3x$

2. Differentiate: $u' = 2x$ and $v' = 3$

3. Apply formula:

$$f'(x) = \frac{(3x)(2x) - (x^2 + 1)(3)}{(3x)^2}$$

4. Expand and simplify:

$$f'(x) = \frac{6x^2 - 3x^2 - 3}{9x^2} = \frac{3x^2 - 3}{9x^2}$$

5. (Optional) Divide everything by 3:

$$f'(x) = \frac{x^2 - 1}{3x^2}$$

🔴 Hard: Dealing with Roots

Question: Differentiate $f(x) = \frac{\sqrt{x}}{x + 2}$

1. Rewrite: $u = x^{1/2}$ and $v = x + 2$

2. Differentiate: $u' = \frac{1}{2}x^{-1/2}$ and $v' = 1$

3. Apply formula (take a deep breath):

$$f'(x) = \frac{(x + 2)(\frac{1}{2}x^{-1/2}) - (x^{1/2})(1)}{(x + 2)^2}$$

4. Expand the top:

$$f'(x) = \frac{\frac{1}{2}x^{1/2} + x^{-1/2} - x^{1/2}}{(x + 2)^2}$$ $$f'(x) = \frac{-\frac{1}{2}x^{1/2} + x^{-1/2}}{(x + 2)^2}$$

(In an exam, stopping here is usually perfectly fine unless asked to simplify further!)
