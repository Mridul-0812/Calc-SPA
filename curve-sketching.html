Applications / Curve Sketching

The Art of Curve Sketching

Calculus allows us to "see" a function without graphing software. By analysing the first and second derivatives, we can pinpoint exactly where a curve turns, where it accelerates, and how it behaves.

Visual Concept: The Anatomy of a Curve

Geometry Scanner

Every smooth curve is built from two fundamental properties:

  • Gradient ($f'$): Is the curve going up or down? When it is flat ($f' = 0$), we have a Stationary Point.
  • Concavity ($f''$): Is the curve bending upwards (like a smile) or downwards (like a frown)? A change in concavity is a Point of Inflection.

The Analytical Toolkit

Calculus Tool Condition Geometric Meaning
First Derivative $f'(x) = 0$ Stationary Point (Tangent is perfectly horizontal)
Second Derivative $f''(x) > 0$ Concave Up ∪ (Indicates a Local Minimum)
Second Derivative $f''(x) < 0$ Concave Down ∩ (Indicates a Local Maximum)
Second Derivative $f''(x) = 0$ Possible Point of Inflection (Must check concavity change!)

Worked Examples

01

Finding and Classifying Stationary Points

Analyse the curve $y = x^3 - 3x^2 - 9x + 5$.

Step 1: Find the first derivative.
$y' = 3x^2 - 6x - 9$
Step 2: Solve $y' = 0$ to find stationary points.
$3(x^2 - 2x - 3) = 0$
$3(x - 3)(x + 1) = 0 \implies x = 3, x = -1$
Step 3: Use the Second Derivative Test ($y''$).
$y'' = 6x - 6$
At $x = 3$: $y'' = 6(3) - 6 = 12 > 0$ (Concave Up $\implies$ Minimum)
At $x = -1$: $y'' = 6(-1) - 6 = -12 < 0$ (Concave Down $\implies$ Maximum)
Step 4: Find $y$-coordinates.
Substitute $x = 3$ and $x = -1$ back into the original equation.
$f(3) = -22$ and $f(-1) = 10$.
Local Maximum at $(-1, 10)$
Local Minimum at $(3, -22)$
02

Finding Points of Inflection

Find the point of inflection for $y = x^4 - 6x^2 + 4$.

Step 1: Find the second derivative.
$y' = 4x^3 - 12x$
$y'' = 12x^2 - 12$
Step 2: Solve $y'' = 0$.
$12(x^2 - 1) = 0 \implies x = 1, x = -1$
Step 3: Verify concavity changes.
For $x = 1$: Check $x=0$ ($y''=-12$) and $x=2$ ($y''=36$). Concavity changes!
For $x = -1$: Check $x=-2$ ($y''=36$) and $x=0$ ($y''=-12$). Concavity changes!
Points of Inflection at $x = 1$ and $x = -1$.

Practice Challenges

Level: Foundation Q1

Find the coordinates of the stationary point for the parabola $y = x^2 - 8x + 15$ and state its nature.

Set $y' = 0$. Once you find $x$, plug it back into the original $y$ equation. Check $y''$ to see if it is a max or min.

$y' = 2x - 8 = 0 \implies x = 4$.

$y(4) = (4)^2 - 8(4) + 15 = -1$.

$y'' = 2$, which is $> 0$, so it is a minimum.

Final: Minimum at $(4, -1)$

Level: Advanced Q2

Show that $y = e^x - x$ has exactly one stationary point and determine its nature.

Differentiate to get $y' = e^x - 1$. When does $e^x - 1 = 0$? Remember your logarithm rules.

$y' = e^x - 1 = 0 \implies e^x = 1 \implies x = 0$.

$y(0) = e^0 - 0 = 1$. So the point is $(0, 1)$.

$y'' = e^x$. At $x=0$, $y'' = e^0 = 1 > 0$ (Minimum).

Final: Minimum at $(0, 1)$

Level: Extension Q3

The curve $y = x^3 + ax^2 + bx$ has a stationary point at $(1, 4)$. Find the values of $a$ and $b$.

You have two pieces of information:
1. The curve passes through $(1, 4)$, so $y(1) = 4$.
2. It is a stationary point at $x=1$, so $y'(1) = 0$.
Set up simultaneous equations.

From $y(1) = 4$:
$1^3 + a(1)^2 + b(1) = 4 \implies a + b = 3$ (Equation 1)

From $y'(1) = 0$:
$y' = 3x^2 + 2ax + b$
$3(1)^2 + 2a(1) + b = 0 \implies 2a + b = -3$ (Equation 2)

Subtract (1) from (2): $a = -6$. Substitute back to find $b = 9$.

Final: $a = -6, b = 9$
