The Grand Table of Derivatives

The definitive repository for HSC Advanced and Extension 1. All trigonometric functions are defined in radians.

Standard Trigonometric Functions

Function $f(x)$ Derivative $f'(x)$ Chain Rule Form: $\frac{d}{dx}[g(f(x))]$
$\sin x$ $\cos x$ $f'(x)\cos(f(x))$
$\cos x$ $-\sin x$ $-f'(x)\sin(f(x))$
$\tan x$ $\sec^2 x$ $f'(x)\sec^2(f(x))$
$\csc x$ $-\csc x \cot x$ $-f'(x)\csc(f(x))\cot(f(x))$
$\sec x$ $\sec x \tan x$ $f'(x)\sec(f(x))\tan(f(x))$
$\cot x$ $-\csc^2 x$ $-f'(x)\csc^2(f(x))$

Inverse Trigonometric Functions Ext 1

Function Derivative $\frac{dy}{dx}$ Domain Constraints
$\sin^{-1} \frac{x}{a}$ $\frac{1}{\sqrt{a^2 - x^2}}$ $-a < x < a$
$\cos^{-1} \frac{x}{a}$ $-\frac{1}{\sqrt{a^2 - x^2}}$ $-a < x < a$
$\tan^{-1} \frac{x}{a}$ $\frac{a}{a^2 + x^2}$ All real $x$

Algebraic, Exponential & Logs

Function Derivative Logic Note
$x^n$ $nx^{n-1}$ General Power Rule
$e^{f(x)}$ $f'(x)e^{f(x)}$ Base $e$ remains unchanged
$\ln(f(x))$ $\frac{f'(x)}{f(x)}$ Log of a function
$a^{x}$ $a^{x} \ln a$ General Exponential (Ext 1)
$\log_a x$ $\frac{1}{x \ln a}$ Change of base required

Formula Practice

What is the derivative of $y = \csc(3x)$?

Using the chain rule and the table above:

$\frac{dy}{dx} = -3 \csc(3x) \cot(3x)$
